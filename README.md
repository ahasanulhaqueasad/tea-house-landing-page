# 🍵 Tea House - Modern Beverage Web Application

A sleek, responsive, and modern frontend web application designed for a premium tea house/cafe, built with **Tailwind CSS**.

## ✨ Features

* **Hero & Featured Products:** Eye-catching product showcase grid displaying various premium teas (Milk Tea, Black Tea, Green Tea, Lemon Tea) with smooth hover effects.
* **Brand Story Section:** Clean layout highlighting brand uniqueness and premium quality.
* **Customer Testimonials:** Interactive super client review cards with an overlapping layout.
* **News & Events:** Latest blog/events section featuring discount updates and announcements.
* **Fully Responsive:** Optimized for mobile phones, tablets, and desktop screens using Tailwind CSS grid and flexbox.

## 🛠️ Built With

* **HTML5**
* **Tailwind CSS** (via CDN)
* **Google Fonts / Custom SVGs**
